HTML

  • The head element contains information about the webpage, but cannot be seen by the user
  • The body element represent the visible content shown to the user.
  • Some of the basic elements used to provide context and framework are: Text, Image, Header, and List elements.
  • There are other elements that HTML uses to describe the browser, such as "Meta-related", "Content division", and "Semantic" elements.
  • HTML elements are composed of angle brackets ("< >").

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside the element.
  • Size of the padding is proportionate to the size of the element.
  • There are three types of ways to style a page using CSS: Inline CSS, Internal CSS, and External CSS.
  • External CSS is the most common and versatile way to style a webpage because changes can be easily made externally.
  • In order to incorporate the external CSS file, you must use the link the external source to your HTML file.
  • The CSS Rule is comprised of a selector, declaration block, property, and value.
  • The Selector defines the element or attributes to which to the rules, or declarations, will apply.
  • Declarations contain two important componenets, the property and the value.
  • The property is the type of style for a given element or attribute.
  • The value is the specific characteristic for the given property.
