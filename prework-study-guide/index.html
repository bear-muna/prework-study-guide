HTML

  • The body element represent the visible content shown to the user.

  • HTML elements are composed of angle brackets ("< >").

Git

  • Enter your Git notes here
  • Git or GitBash is a developer tool to quickly navigate the files within your machine
  • In order to perform a Git function, you must preface with "git"
  • git status: Check what branch we are currently on
  • git checkout -b branch-name: Creates a new branch and switches to it
  • git add -A: Adds all modifications from the local environment to the staging area
  • git commit -m mod-explanation: Commits/saves currently staged work and the branch will be updated
  • git pull origin main: Quick and easy way to make sure that modifications made onto current code is the up to date code
  • git push origin branch-name: Pushes/merges changes from the local environment onto the GitHub branch
